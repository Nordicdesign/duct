Form elements

Last modified: 12/11/2014

Every element must be part of a fieldset, otherwise the styles won't be rendered appropriately (accessiblity considerations aside).

	
<form>
  <fieldset>
    <p>
      <label>This is the label</label>
      <input type="text" value="Some predefined text here">
    </p>
  </fieldset>
</form>

Buttons

Normal button

			
<input class="button" value="Normal Button">

Primary action button

			
<input class="button primary" value="Primary action">

Disabled button

			
<input class="button disabled" disabled value="Disabled">

Submit button

			
<input type="submit" value="Submit button">
