Responsive grid

Last modified: 13/11/2014

The responsive grid is key in the creation of a responsive website. We use 8 columns in large screens, and 4 columns in small devices, with a gutter of 10px. This can be configured in the _config.scss file.

The site has only one breakpoint at 55em (550px).

Classes largeN will be active at screens bigger than 550px, and the smallN below that figure, allowing you to have different layouts for the same page, on large and small devices.

Example of grid for a large device only

	
<div class="row">
  <div class="large8 col">8</div>
</div>
<div class="row">
  <div class="large1 col">1</div>
  <div class="large7 col">7</div>
</div>
<div class="row">
  <div class="large2 col">2</div>
  <div class="large5 col">5</div>
  <div class="large1 col">1</div>
</div>
