Breadcrumb

Last modified: 13/11/2014

With the exception of the homepage, the breadcrumb block should be on every page to help visitors navigate the page, and understand the content's hierarchy.

How it looks like

  • Home
  • Category
  • Current page

HTML

   	

<nav class="breadcrumb">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Category</a></li>
    <li><strong>Current page</strong></li>
  </ul>
</nav>

SCSS

	

/*BREADCRUMB*/
nav.breadcrumb {
	font-size: 1.4rem;
	color: #3A8EE2;
	margin-bottom: 2em;

	ul li {
		float: left;
		list-style: none;
	}

	a {
		margin: 0 5px;
		text-decoration: none;
	}
	a:first-child {
		margin-left: 0;
	}

	a:hover, a:focus {
		color: $primary-color;
	}

	strong {
		font-weight: bold;
		color: $primary-color;
	}		

}
